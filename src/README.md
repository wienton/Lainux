## Работа с турбо установщиком LainuxOS

### На данный момент вышла только бета версия, версия 0.1, которая не представляет из себя топового установщика

> Могут быть баги ;)

Давайте разберемся как запустить и работать с установщиком на данный момент.

В директории bin у нас лежат два файла, нам нужны turbo_lainux

Запускаем его через команду, поэтапно

```bash

chmod +x ./turbo_lainux

./turbo_lainux

```
Воу-ля, у нас открылся интефрейс установщика, дальше мы можем установить на виртуальную машину(((ЛУЧШЕ ВСЕГО!!!!)))
