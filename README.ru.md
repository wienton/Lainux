**Lainux** — это высокоспециализированный дистрибутив Linux, ориентированный на командную строку, тщательно разработанный на основе надежной платформы Arch Linux. Разработанный **wienton**, этот проект направлен на расширение границ системной производительности, абсолютного контроля и бескомпромиссной безопасности для опытных пользователей, разработчиков систем и энтузиастов низкоуровневого программирования.

## Обзор проекта
Lainux стремится обеспечить облегченную, эффективную и прозрачную операционную среду за счет значительного сокращения системных накладных расходов и абстрагирования ненужных компонентов. Методология разработки основана на глубоком понимании базовой архитектуры системы, использовании **C** для разработки критически важных утилит и **NASM (x86 Assembler)** для точного, чувствительного к производительности кода, где это применимо.


Наша основная философия подчеркивает:
*   **Экстремальная производительность:** достигается за счет высокооптимизированного пользовательского ядра и строго отобранного набора активных служб.
*   **Детерминизм:** обеспечивает беспрецедентный контроль над поведением системы и распределением ресурсов.
*   **Безопасность по дизайну:** минимальная поверхность атаки благодаря компактному программному обеспечению и прозрачным системным процессам.
*   **Расширяемость:** модульная архитектура, разработанная для гибкого расширения без лишних нагрузок.

## Ключевые особенности и отличительные черты
* **Профессиональный установщик TUI:** полностью интерактивный установщик с клавиатурным управлением, созданный с помощью ncurses, предлагающий выбор диска, обнаружение UEFI/BIOS, безопасное стирание, сводку по оборудованию и развертывание виртуальной машины — и все это без выхода из терминала.
* **Отсутствие лишних компонентов на рабочем столе:** нет дисплей-менеджера, нет фоновых демонов, нет телеметрии. То, что вы устанавливаете, — это именно то, что работает. Полная прозрачность системы — от init до пользовательского пространства — гарантирована конструкцией.
* **Воспроизводимая система сборки:** генерация ISO, компиляция ядра и упаковка пакетов полностью скриптованные в POSIX-совместимом Bash. Воспроизводимые сборки обеспечиваются с помощью источников с проверенными контрольными суммами и детерминированных флагов компилятора (-ffunction-sections -fdata-sections -Wl,--gc-sections).
* **Специально разработанное ядро Linux:** специально адаптировано для Lainux, скомпилировано с учетом требований к производительности и безопасности. Это включает в себя точную настройку параметров ядра, удаление ненужных модулей и оценку патчей реального времени (RT) или улучшенных патчей безопасности, где это применимо.
* **Минималистичная система инициализации:** использует *[например, systemd с сильно сокращенным набором модулей или быструю альтернативу, такую как Runit/OpenRC, если она принята]* для быстрой загрузки и незначительного потребления ресурсов.
*   **Минималистичная графическая среда (опционально):** Разработана для беспроблемной интеграции с высокоэффективными оконными менеджерами (*например, i3wm, dwm, bspwm*) или для работы исключительно в интерфейсе командной строки.
*   **Пользовательская библиотека/утилиты C:** Разработка пользовательских системных утилит, критичных для производительности, и API на C, обеспечивающих прямое и эффективное взаимодействие с аппаратными и системными ресурсами.
*   **Надежная автоматизация Bash:** Комплексные скрипты для оптимизации развертывания системы, управления конфигурацией и настройки после установки.
*   **Целевая оптимизация:** использование низкоуровневых знаний из Assembler для выявления и оптимизации узких мест в критических системных путях.


## Целевая аудитория и области применения
Lainux предназначен для профессионалов и опытных пользователей в таких областях, как:
*   **Разработка встроенных систем и Интернет вещей:** легкая и управляемая платформа для разработки специализированного прошивка и операционных систем.
*   **Системный анализ и обратное проектирование:** обеспечивает чистую среду для разработки эксплойтов, анализа вредоносных программ и исследований в области безопасности.
*   **Высокопроизводительные вычисления (HPC):** идеально подходит для тестов и рабочих нагрузок, требующих максимального использования аппаратных ресурсов.
*   **Академические и исследовательские проекты:** незаменимый инструмент для углубленного изучения внутреннего устройства Linux и дизайна операционных систем.


## Статус проекта и план действий
Lainux находится в стадии активной разработки. Текущие этапы разработки, осуществляемые под руководством `wienton`, включают:
*  Усовершенствование конфигураций сборки ядра и интеграцию патчей.
*  Реализация `lainux-coreutils` (набор основных системных утилит на языке C).
*  Создание
стабильной минимальной среды командной строки.
*  Разработка надежного конвейера генерации ISO для тестируемых выпусков.

Информация о ходе работ, история коммитов и особенности ветвей открыто доступны в этом репозитории.


## Технологический стек
*   **C/C++:** Основной язык для системных компонентов, утилит и взаимодействия с ядром.
*   **NASM / GNU Assembler:** Для микрооптимизации, анализа загрузчика и прямого взаимодействия с оборудованием.
*   **Скрипты Bash:** необходимы для автоматизации, инициализации системы и управления пакетами.
*   **Arch Linux Base:** обеспечивает гибкую и современную основу экосистемы.
