# Работа с установщиком Turbo Lainux (Beta v0.1)

На текущем этапе разработки представлена версия *0.1 (Beta)*. Данная итерация является ранним прототипом установщика и предназначена для ознакомления и тестирования основного функционала.

> *Примечание:* В связи со статусом бета-версии возможны нестабильная работа и наличие багов. Рекомендуется использовать для тестов изолированные среды.

---

## Быстрый запуск

Все необходимые исполняемые файлы расположены в директории `bin`. Основным компонентом является бинарный файл `turbo_lainux`.

### Инструкция по запуску:

Для запуска процесса установки последовательно выполните следующие команды в терминале:

1. *Предоставление прав на исполнение:*

```bash
   chmod +x ./turbo_lainux
```
2. **Запуск установщика:**
 ```bash
   ./turbo_lainux
```

---

## Работа в режиме виртуализации (Рекомендуется)

На данном этапе разработки *настоятельно рекомендуется* производить установку внутри виртуальной машины. Установщик LainuxOS обладает встроенной логикой автоматизации для работы с виртуальными средами.

### Алгоритм работы установщика:
1. *Детекция окружения:* Система автоматически определяет параметры вашего хоста.
2. *Развертывание гипервизора:* Установщик самостоятельно настраивает среду QEMU.
3. *Подготовка ресурсов:* В автоматическом режиме происходит генерация конфигурационных файлов (cfg) и удаленная загрузка актуального ISO-образа.
4. *Сборка системы:* Автоматическая инициализация и запуск виртуальной машины с предустановленными параметрами.
